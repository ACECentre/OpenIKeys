Attaching Overlays to Applications

If you are using IntelliKeys USB with a Windows operating system or Macintosh
OSX (10.5 and above), the Control Panel includes an Applications tab that
allows you to attach overlays to most applications on your system. Once you have
attached an overlay to an application, that overlay is sent whenever the
application is active.
