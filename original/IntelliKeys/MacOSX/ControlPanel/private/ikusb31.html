IntelliKeys USB Control Panel Changes in Version 3.1

The information below supplements the existing IntelliKeys USB User's Guide:

-- The Advanced tab is renamed the Mode tab. However, the functionalities remain essentially the same.

The Mode (formerly Advanced) tab of the IntelliKeys USB control panel includes the words

IntelliKeys and IntelliSwitch functions will be determined by...

o        Overlays

o        One overlay only

o        Switch Presets

o        DiscoverPro

Select the option that best completes the sentence for your situation.

… Overlays

This option allows custom overlay sending by applications (including ClickIt!*) and by IntelliKeys USB via the Applications tab in the IntelliKeys USB and IntelliSwitch Control Panel. See Chapter 4 for the details of the Applications tab features.

...One overlay only

Select this option if you want to use only one custom overlay for all applications on the computer. This setting overrides any last sent overlays.

...Switch Presets

Select this option if you wish to restrict the use of IntelliKeys USB or IntelliSwitch to one of the switch presets in the drop-down list box. This setting overrides any last sent overlays.

Selecting this option has the following results:

IntelliKeys USB will function as a two-switch device. As a two-switch device, the entire left side of IntelliKeys USB functions as one key and the entire right side of IntelliKeys USB functions as a second key. Selections in the drop-down list box assign the first key function listing to the left side of IntelliKeys USB and the second key function to the right side of IntelliKeys USB.

IntelliKeys USB switch ports will function as switches 1 and 2. The first key function listing is assigned to switch port one and the second key function to switch port two.

IntelliSwitch will be assigned switches 1 and 2 as well as additional switch settings for 3, 4 and 5

...DiscoverPro

This option only appears below the Switch Preset option when you have Discover Pro 2.0.2 installed on your computer. Select this option to use DiscoverPro setups for your IntelliSwitch or IntelliKeys USB.

Note1: DiscoverPro has been removed from the Applications tab.

Note2: To use ClickIt! you must not have DiscoverPro 2.0 installed on the same computer.

Select "But still allow applications to send their own overlays" to enable DiscoverPro to use attached overlays (when present) instead of the attached DiscoverPro setup. For example, to allow DiscoverPro to use Classroom Suite overlays instead of the DiscoverPro ICS Start SUS setup, select this option.  

-- The IntelliKeys USB task bar pop up menu has been updated to reflect the Mode tab and the updated Applications tab.

-- The IntelliKeys USB Applications tab has been simplified by moving selections for DiscoverPro and ClickIt! to the Mode tab as described above. "The Control Panel will not send an overlay" replaces the text "the Control Panel will send no overlay", but the functionality is the same -- that the IntelliKeys USB control panel will not send an overlay for the selected application. This still allows applications that can send their own overlays (including ClickIt!) to continue to do so. 

-- Switch Preset 26 Mouse Directions has been added:

Mouse Up, Mouse Down, Mouse Left, Mouse Right, Button Click
