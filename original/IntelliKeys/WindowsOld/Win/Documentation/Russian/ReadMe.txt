IntelliKeys USB
Версия 3.5.1
Важная информация


1 - Что изменилось в этой версии

Версия 3.5.1 включает поддержку Windows 8 и Mac OS 10.8 (Mountain Lion) в
дополнение к улучшенной поддержке 64-разрядной операционной системы Windows
для международных версий.


2 - Замечания по продукту

Замечания по выполнению установки.
Для установки программного обеспечения IntelliKeys USB необходимо иметь права
администратора на компьютере. Перед установкой этой версии отсоедините
устройство IntelliKeys USB от компьютера. Затем удалите старое программное
обеспечение IntelliKeys USB.

Чтобы удалить предыдущую версию, выполните следующие действия.
- В Windows XP используйте элемент «Установка и удаление программ» на Панели
  управления.
- В Windows 7 и Windows 8 используйте функцию «Программы и компоненты».
- В ОС Mac используйте программу удаления, входящую в комплект установки.

Однако если вы пользуетесь Popular Software Overlays, НЕ УДАЛЯЙТЕ предыдущую
версию программного обеспечения IntelliKeys USB!

Очень важная информация для пользователей Mac!

- Версия 3.5.1 будет работать только на системах Intel с операционной
  системой Mac OS 10.5.8 - 10.8. Установщик находится в «Installers/V3.5
  (10.x, Intel Only)» в версии продукта на английском языке, и он является
  единственной версией установщика для международных версий продукта.

- В системе Mac OS X значок IntelliKeys USB отображается на панели меню,
  обычно рядом с часами. Выберите этот значок, чтобы открыть панель
  управления IntelliKeys USB. В классическом режиме на панели меню выберите
  Finder, чтобы выбрать значок IntelliKeys USB для открытия панели управления
  IntelliKeys USB.

- Обычная правая кнопка мыши оверлея IntelliKeys USB будет работать в
  программах Mac OS X, которые поддерживают функцию щелчка правой кнопкой
  мыши.

- В системах Mac каждая подсоединенная клавиатура распознается как отдельная
  и уникальная. Это означает, что световые индикаторы Num Lock и Caps Lock на
  IntelliKeys USB будут отражать состояние только IntelliKeys USB.
  Синхронизация этих световых индикаторов с индикаторами на стандартной
  клавиатуре не требуется.


3 - Поддержка операционных систем

IntelliKeys USB версии 3.5 будет работать в следующих системах:

- системы Intel с ОС Mac версий от 10.5 до 10.8;
- Windows: 32- и 64-разрядные ОС Windows XP, Windows Vista, Windows 7 и
  Windows 8.
