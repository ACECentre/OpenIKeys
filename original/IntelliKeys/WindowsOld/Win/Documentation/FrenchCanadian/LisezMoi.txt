IntelliKeys USB
Version 3.5.2
Renseignements importants


1 – Changements apportés dans cette version

La version 3.5.2 inclut la prise en charge de Windows 8 et de Mac OS 10.8
(Mountain Lion) en plus de la prise en charge du système d’exploitation Windows
à 64 bits amélioré pour les versions internationales.


2 - Notes sur les produits

Note d’installation :
Pour installer le logiciel IntelliKeys USB, vous devez avoir un accès
d’administrateur pour l'ordinateur. Avant d’installer la nouvelle version,
débranchez IntelliKeys USB de l'ordinateur. Désinstallez ensuite l’ancien
logiciel IntelliKeys USB.

Pour désinstaller l’ancienne version :
- Pour Windows XP, utilisez Ajout/Suppression de programmes à partir du Panneau
  de configuration.
- Pour Windows 7 et 8, utilisez Programmes et fonctionnalités.
- Pour le Mac, utilisez le programme Désinstaller fourni avec l’installation.

Toutefois, si vous avez Popular Software Overlays, NE désinstallez PAS l’ancien
logiciel IntelliKeys USB!

Renseignements très importants pour les utilisateurs du Mac !

- La version 3.5.2 ne fonctionne que sur les systèmes Intel exécutant Mac OS
  10.5.8 - 10.8. Cela peut se trouver dans « Installers/V3.5 (10.x, Intel
  Only) » de la version du produit en anglais et c’est la seule version
  d’installation pour les versions internationales du produit.

- Sur le Mac OS X, l’icône IntelliKeys USB s'affiche sur la barre de menus,
  habituellement à côté de l'affichage de l'heure. Sélectionnez l’icône pour
  ouvrir le panneau de configuration de IntelliKeys USB. En mode classique, à
  partir de la barre de menus, sélectionnez Localisateur puis cliquez sur
  l’icône IntelliKeys USB pour ouvrir le panneau de configuration IntelliKeys
  USB.

- La touche du clic droit de la souris de superposition IntelliKeys USB
  standard fonctionne avec les programmes de Mac OS X prenant en charge la
  fonctionnalité du clic à droite.

- Chaque clavier connecté sur un Mac est reconnu en tant que périphérique
  unique et distinct. Cela signifie que les voyants Verr. Num. et Verr. Maj.
  sur IntelliKeys USB indiqueront l'état du IntelliKeys USB uniquement. Ces
  voyants n’auront pas à être synchronisés avec les voyants du clavier
  standard.


3 – Prise en charge de système d’exploitation

La version 3.5 de IntelliKeys USB fonctionne sur

- Les systèmes Intel exécutant Mac 10.5 à 10.8
- Windows à 32 bits et à 64 bits, Windows XP, Windows Vista, Windows 7 et
  Windows 8.

Si vous possédez une version antérieure de IntelliKeys USB, veuillez visiter
notre site Web pour consulter les détails techniques : www.intellitools.com
