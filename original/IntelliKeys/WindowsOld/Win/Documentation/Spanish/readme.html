IntelliKeys USB
Versión 3.5.2
Información importante


1 - Qué cambió en esta versión

La versión 3.5.2 cuenta con soporte para Windows 8 y Mac OS 10.8 (Mountain
Lion), además de soporte mejorado en el sistema operativo Windows de 64 bits
para versiones internacionales.


2 - Notas del producto

Nota sobre la instalación:
Para instalar el software IntelliKeys USB es necesario tener acceso de
administrador a la computadora. Antes de instalar esta nueva versión,
desconecte el IntelliKeys USB de la computadora. Después desinstale el software
anterior de IntelliKeys USB.

Para desinstalar la versión anterior:
- En Windows XP, use Agregar o quitar programas en el Panel de control.
- En Windows 7 y 8, use Programas y características.
- En Mac, use el programa de desinstalación que viene con la instalación.

Sin embargo, si usted tiene Popular Software Overlays, NO DESINSTALE la versión
anterior de IntelliKeys USB.

Información muy importante para usuarios de Mac

- La versión 3.5.2 solo funciona en sistemas Intel que operen con Mac OS
  10.5.8 - 10.8. Esto puede encontrarse en “Installers/V3.5 (10.x Intel Only)”,
  en la versión en inglés del producto, y es la única versión del instalador
  para versiones internacionales del producto.

- En Mac OS X, el icono de IntelliKeys USB aparece en la barra de menús, por lo
  general junto al reloj. Seleccione ese icono para abrir el panel de control
  de IntelliKeys USB. En modo clásico, en la barra de menús seleccione Finder
  para seleccionar el icono de IntelliKeys USB y abrir el panel de control
  correspondiente.

- El clic derecho estándar del ratón de IntelliKeys USB Overlays funciona en
  los programas de Mac OS X que soporten la función de clic derecho del ratón.

- En las Mac, cada teclado conectado se reconoce como independiente y único.
  Esto significa que las luces de Bloqueo numérico y Bloqueo de mayúsculas en
  IntelliKeys USB informan el estado únicamente de IntelliKeys USB. Esas luces
  no necesitan sincronizarse con las luces del teclado estándar.


3 - Compatibilidad con sistemas operativos

IntelliKeys USB, versión 3.5, funciona en:

- Sistemas Intel que ejecuten Mac 10.5 a 10.8.
- Windows XP de 32 y 64 bits, Windows Vista, Windows 7 y Windows 8.

Si tiene una versión anterior de IntelliKeys USB, consulte los detalles
técnicos en nuestro sitio web: www.intellitools.com
